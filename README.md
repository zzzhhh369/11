## 📋 项目概述

本项目使用 Python 对某电商平台的商品销售数据进行多维度数据分析。数据来源于一个包含产品信息表和销售数据表的 Excel 工作簿，涵盖 2022 年全年（1-12月）共 40 款产品、2,055 条销售记录。

通过 Pandas 进行数据处理，Matplotlib 进行可视化，分析维度包括：
- 月度销售额统计与环比分析
- 热销产品月度销售趋势
- 产品大类月度销售对比
- 产品小类全年销售额对比

最终为电商运营和营销策略提供数据支持。

## 📊 数据说明

| 数据表 | 字段 | 说明 |
|--------|------|------|
| 信息表 | 产品大类、产品小类、产品名称、产品号、产品售价 | 40条产品基础信息 |
| 销售数据表 | 销售日期、销售单号、产品号、销售数量 | 2055条销售明细记录 |

## 🔧 分析方法

### 1. 数据读取与合并
- 使用 `pandas.read_excel()` 读取两个工作表
- 使用 `pandas.merge()` 按 **产品号** 进行内连接合并

### 2. 特征工程
- **销售金额** = 产品售价 × 销售数量
- **月份** = `df['销售日期'].dt.month` 从日期中提取

### 3. 分组统计
- 使用 `groupby('月份')['销售金额'].sum()` 按月汇总
- 使用 `groupby(['月份', '产品大类'])['销售金额'].sum()` 分类汇总
- 使用 `groupby('产品小类')['销售金额'].sum()` 按小类汇总

### 4. 可视化分析
- **折线图（plot）**：展示 TOP10 产品月度销售趋势
- **分组柱状图（bar）**：对比不同产品大类的月度销售额
- **横向柱状图（barh）**：展示各产品小类的全年销售额占比

## 📈 分析结论

### 数据概览
- 全年销售总额：**5,280,308 元**
- 月均销售额：约 440,026 元
- 数据覆盖 **2个大类**（运动套装、运动配件）和 **6个小类**（上衣、短裤、长裤、袜子、发带、护腕）

### 主要发现

1. **月度趋势稳中有升**：
   - 1月销售额最低（344,878元），7月达到峰值（490,090元）
   - 下半年整体销售额高于上半年，Q4 表现尤为突出

2. **品类差异显著**：
   - 运动套装类产品（上衣、短裤、长裤）贡献了绝大部分销售额
   - 运动配件类（袜子、发带、护腕）单价低但销量大

3. **头部产品集中度高**：
   - 销售额 TOP10 产品贡献了主要销售收入
   - 体现了电商 "爆款" 效应

4. **季节性特征明显**：
   - 夏季（6-8月）运动服装销售旺盛
   - 年末（11-12月）也有销售高峰，可能与促销活动有关

### 运营建议
- 针对热销产品优化库存管理，避免断货
- 利用季节性规律，提前布局促销活动
- 关注低销量产品，分析原因并优化或淘汰
- 探索产品组合销售策略，提升客单价
